# Projekt Docker

## Opis projektu
Projekt przedstawia podstawowe wykorzystanie technologii Docker do uruchamiania kontenerów oraz zarządzania obrazami.

## Funkcjonalności
- Budowanie obrazów Docker
- Uruchamianie kontenerów
- Mapowanie portów
- Zarządzanie siecią Docker

## Wykorzystane technologie
1. Docker
2. Linux
3. Nginx

## Informacje dodatkowe
**Docker** umożliwia łatwe uruchamianie aplikacji w izolowanych środowiskach.

*Kontenery* pozwalają uprościć wdrażanie aplikacji oraz ich testowanie.
